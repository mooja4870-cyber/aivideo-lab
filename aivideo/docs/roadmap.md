# Product Roadmap (Post-MVP)

## 1) Admin Dashboard

- Daily/weekly cost tracking by provider (OpenAI, Replicate, storage, egress)
- Revenue tracking by plan and channel
- Margin view per generated video and per user cohort
- Alerting for abnormal failure/cost spikes

## 2) Subtitle Support

- Burned-in subtitles with FFmpeg full build profile
- Korean line wrapping rules + safe-zone presets
- Optional subtitle style presets (brand colors, captions)

## 3) Background Music

- Royalty-safe BGM library integration
- Auto ducking under narration
- Genre presets by business category

## 4) Video Templates

- Industry templates (cafe, beauty, fitness, clinic, retail)
- Template variables: intro hook, CTA, on-screen text style
- A/B test hooks and CTA variants

## 5) 16:9 Landscape Option

- Output profile switch: 9:16 / 1:1 / 16:9
- Layout-safe text regions per aspect ratio
- Thumbnail variant generation for YouTube landscape

## 6) Direct Publishing

- YouTube upload API integration
- TikTok upload pipeline and publish status sync
- Schedule-based auto publish with retry and audit logs
